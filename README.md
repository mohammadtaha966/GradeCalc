# GradeCalc - Professional GPA Calculator

**Created by Studies Made Easier (SME)**  
*Established February 2026*

## ✨ Features

- GPA Calculator - Real-time semester GPA calculation
- CGPA Tracker - Monitor cumulative GPA across all semesters  
- Goal Planner - Calculate required grades to achieve target CGPA
- Analytics - Visual insights with progress tracking
- 100% Private - All data stays on your device
- Works Offline - Progressive Web App functionality

## 🚀 Quick Start

1. Download all HTML files
2. Open `index.html` in your browser
3. Start calculating!

## 📁 Files Included

- index.html - Main GPA Calculator
- cgpa.html - CGPA Tracker
- goals.html - Academic Goals Planner
- analytics.html - Analytics Dashboard
- guides.html - Grading Guides
- about.html - About Page
